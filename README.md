# C# 闯关学习平台

一个面向零基础小白的 C# 闯关式学习平台，可在浏览器内直接运行 C# 代码，无需安装任何环境。

## ✨ 特性

- 🎯 **闯关式学习**：34 关 · 6 阶段 · 13 章，从零基础到能独立编写小型控制台项目
- 💻 **浏览器内运行 C#**：内置轻量 C#→JS 模拟执行器，无需安装 .NET SDK
- 📚 **分阶段课程**：入门基础 → 控制流 → 数据结构 → 方法与面向对象 → 进阶特性 → 实战项目
- 🌗 **暗色 / 亮色双主题**：自动记忆偏好，0.3s 平滑过渡
- 🎌 **专注模式**（F11）：全屏编辑器，虚化背景，Esc 退出
- 📖 **参考答案侧栏**：从编辑器左侧拉出，不遮挡代码，一键复制
- ⌨ **Console.ReadLine 支持**：运行前弹出输入区，可写交互式入门关卡
- ⌨ **键盘快捷键**：Ctrl+Enter 运行 · Alt+←/→ 切换关卡 · Tab 缩进
- 💾 **本地进度存储**：localStorage 自动保存进度与代码历史，无需登录
- 🌐 **纯静态部署**：可直接部署到 GitHub Pages

## 🚀 本地预览

纯静态站点，直接用浏览器打开 `index.html` 即可；若需本地服务器：

```bash
# Python
python -m http.server 8080

# Node.js
npx serve
```

浏览器打开 `http://localhost:8080`

## 📦 部署到 GitHub Pages

1. Fork 或 push 到 GitHub 仓库
2. 仓库 Settings → Pages
3. Source 选择 `main` 分支，目录选 `/ (root)`
4. 等待几分钟，访问 `https://你的用户名.github.io/仓库名/`

## 📖 课程结构

| 阶段 | 章节 | 关卡数 | 内容 |
|------|------|--------|------|
| 入门基础 | 1-3 | 9 | Hello World、变量与数据类型、运算符 |
| 控制流 | 4-5 | 7 | if/switch/三元、for/while/break |
| 数据结构 | 6-7 | 6 | 数组、字符串方法 |
| 方法与面向对象 | 8-11 | 8 | 方法、类、继承、virtual/override、接口 |
| 进阶特性 | 12 | 1 | List 动态列表 |
| 实战项目 | 13 | 3 | 计算器、猜数字、学生管理系统 |

> 共 **34 关**，全部经内置模拟执行器验证通过。

## 🛠 技术栈

- 原生 HTML / CSS / JavaScript，无任何依赖
- 自研 C#→JS 轻量模拟执行器（`js/runner.js`），支持基础语法、类、继承、接口、List 等
- localStorage 进度存储

## ⚠️ 运行器能力边界

内置执行器为正则翻译式模拟器，**不是真正的 .NET 运行时**，支持的语法子集有限（详见 `js/runner.js`）。课程内容已锚定在该子集内，确保每一关的参考答案都能在本平台正确运行。如需学习 LINQ、async/await、泛型方法等进阶特性，建议配合本地安装 .NET SDK。
